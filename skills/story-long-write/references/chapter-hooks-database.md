> 按类型分类，每个提供真实范例 + 可套用模板

## 一、系统激活型钩子

**真实范例**：
> 就在他万念俱灰之际。一个冰冷的机械音，突然在他脑海中响起。叮！【年少轻狂系统，正式启动。】

**结构**：绝望时刻 → 突然声响 → 系统名称

**模板**：
```
就在{主角}{最低谷状态}之际。
{一个感官描写}。
叮！
【{系统名}，正式启动。】
```

**变体**：
- 就在他准备放弃的时候，眼前突然浮现出一行金色大字。
- 脑海中忽然响起一道机械音。
- 手背上浮现出一个奇异的纹路。

## 二、悬念预知型钩子

**真实范例**：
> 我冷眼看着笑得前仰后合的人。笑吧。多笑会儿。过了今天，你们就再也笑不出来了。

**结构**：对方嚣张 → 主角冷观 → 预告反转

**模板**：
```
{主角}{冷静/冷漠地}看着{对方的嚣张行为}。
{一个短句：继续吧/慢慢来/不着急}。
{预告即将发生的反转，不透露细节}。
```

**变体**：
- 他不知道的是，真正的灾难还在路上。
- 但所有人都忽略了一个细节。
- 可她不知道，那个决定，将彻底改变一切。

---

## 三、截断型钩子

**真实范例**：
> 仅仅一口！她的瞳孔猛地收缩成针尖状。轰！一股恐怖到难以形容的热流，瞬间在她的腹中炸开！这......这不是普通的粥！这是什么？

**结构**：小动作 → 巨大反应 → 疑问（不回答）

**模板**：
```
{一个小动作}。
{剧烈反应}。
{震惊/疑问——本章结束，不给答案}。
```

**变体**：
- 她打开门——床上躺着的那个男人，让她的血液瞬间凝固。
- 系统面板上跳出一行字，他看清后，整个人都愣住了。
- 电话那头沉默了三秒，然后说出了一个他做梦都没想到的名字。

## 四、选择型钩子

**真实范例**：
> 【选择一：无敌一百年！】奖励：勾玉轮回眼、查克拉果实……代价：无敌百年后即刻暴毙。
> 【选择二：永生者】奖励：永恒不死！代价：实力永久卡在超影级。

**结构**：两个/多个选项 + 各自利弊 → 章节结束

**模板**：
```
【选择一：{诱人选项}】
奖励：{极其诱人的奖励}
代价：{隐含风险}
【选择二：{另一个选项}】
奖励：{另一种诱人奖励}
代价：{另一种风险}
```

**变体**：
- 眼前出现两条路：一条通往光明，一条通往深渊。
- "你只有一次机会。"他看着面前的两个按钮。

## 五、危机升级型钩子

**真实范例**：
> "都给老子出来！快点！"哐啷！！！一只沾满泥污的军靴，粗暴地踹开了笼门。

**结构**：突然声响 → 新威胁出现

**模板**：
```
{突然的声响/动作}！
{新威胁出现}。
{主角的恐惧/紧迫反应}。
```

**变体**：
- 走廊尽头传来脚步声。越来越近。
- 手机屏幕突然亮了。一条来自未知号码的消息。
- 门把手在转动。

---

## 六、身份即将揭露型钩子

**真实范例**：
> 一个富商盯着我，迟疑的开口："沈大姑娘可曾去过西域？""我在那边经商，听说西域那边有个女人，跟皇室有些瓜葛，也叫沈什么月。"

**结构**：旁人发现线索 → 主角紧张 → 暂时被打断

**模板**：
```
{一个旁观者}突然{提出疑问/发现线索}。
"{暗示主角真实身份的话}"
{其他人不以为意，主角暗暗紧张}。
```

## 七、情感转折型钩子

**真实范例**：
> 就在这时，江亦瑶的表情突然变了，眼眶一红，两行清泪毫无征兆地滚落下来。她带着哭腔："求你了，苏牧，让我走吧……"

**结构**：情绪突变 → 意想不到的示弱 → 主角动摇

**模板**：
```
就在这时，{角色}的{表情/态度}突然{大变}。
{一个出人意料的示弱/请求动作}。
"{戳中主角软肋的话}"
```

## 八、信息差型钩子

**真实范例**：
> 花有重开日，人无再少年。自己怎么就把日子，过成了这副狗屎样？

**结构**：主角自嘲/感慨 → 读者知道主角即将迎来转机 → 章末反转

**模板**：
```
{主角的自我怀疑/感慨}。
{一个意象/比喻}。
{暗示即将改变，但不明说}。
```

## 九、对话悬念型钩子

**真实范例**：
> "我有一个请求。"苏牧缓缓说道。
> "你说。"
> "离婚之前，我要证明一次自己。"

**结构**：提出请求 → 对方答应 → 说出意想不到的话

**模板**：
```
"{提出请求的前半句}"
"{对方回应}"
"{意想不到的后半句——不是读者预期的}"
```

**模板**：

```
{出人意料的结果}。
{暗示过程的一句话}。
```

## 章节钩子选择指南

| 故事阶段 | 推荐钩子类型 | 频率 |
|----------|-------------|------|
| 第1章 | 系统激活/截断/危机升级 | 必须强 |
| 2-3章 | 选择/信息差/情感转折 | 强 |
| 中期日常 | 对话悬念/信息差/身份即将揭露 | 中 |
| 高潮前 | 危机升级/悬念预知 | 强 |
| 大结局 | 时间跳跃/情感转折 | 收束 |
